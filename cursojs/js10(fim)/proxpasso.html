Próximos Passos!

Sobre os próximos passos da linguagem e estudos.

A princípio, o próximo passo é estudar muito HTML, CSS e os fundamentos do JavaScript para melhorar a base e a lógica, consequentemente melhorando como programador.

Também é necessário dizer que vamos ter que treinar muito a lógica de programação e fazer exercícios de lógica para conseguir tirar de letra estruturas simples e um pouco mais complexas em questão de lógica e matemática.

JavaScript e sua sequência.

O primeiro e mais importante passo é aprender melhor sobre functions, como funcionam e suas potencialidades. Dominar uma function é essencial para dominar o JavaScript e as aplicações Web.

O JavaScript é uma linguagem funcional que trabalha com objetos então é muito interessante entender e aprender sobre objetos, como funcionam e como trabalhar com eles para melhor utilização da linguagem e extrair seu real conteúdo.

Também é importante aprender sobre modularização, para que serve e como aplicar. Tomando como base que modularizar é necessário para reutilizar códigos e deixá-los separados em arquivos separados.

Temos, ainda, a necessidade de estudar RegEx, abreviação para expressões regulares que o JavaScript também utiliza e trabalha para validar dados. Economiza muito tempo em código.

Por fim, para esse curso, temos ainda um pequeno conjunto de tecnologias que também são necessárias para se aprender e dominar no mundo da programação front-end. A tecnologia do Json, conjunto / estrutura de objetos que é possível transmitir e salvar dados localmente, é de extrema importância para se aprender. Estudar e compreender o Ajax e suas funcionalidades. Assim como não se limitar apenas ao Client Side do JavaScript e procurar entender e se aprofundar no NodeJS é de extrema importância.
