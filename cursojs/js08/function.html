Funções!

Sobre chamadas e funções.

A respeito das funções, podemos dizer que toda função tem os seguintes fatores: uma chamada, um conjunto de parâmetros, uma ação e um retorno. Nem sempre a função vai ter um conjunto de parâmetros ou retorno.

Também podemos dizer, tecnicamente, que uma função é: são ações executadas assim que são chamadas ou em decorrência de algum evento.

Uma função pode receber parâmetros e retornar um resultado

Sequência de exemplos práticos.

Durante os exemplos, temos alguns fatores relevantes a serem levantados.

O primeiro exemplo é uma function com a capacidade de retornar se um número é par ou impar.

O segundo exemplo é uma function simples com o intuito de retornar a soma entre dois números, atentando-se que, caso não sejam especificados quais são os valores de n1 e/ou n2, ele entenderá como 0

O terceiro exemplo cria uma variável que recebe uma function X e pede para a function retornar com o valor de x*2, nisso a function retorna o valor de x vezes 2, levando em consideração que x receba 5.

O quarto exemplo cria uma function que quer me retornar o fatorial de algum número, no nosso exemplo é o número 5, então eu crio um for de repetição para achar o fatorial e depois indico o número 5 no log, encerrando a aplicação com ele me retornando o fatorial de 5 como esperado.

O quinto e último exemplo trata-se de uma forma recursiva de criar o exemplo 4 fazendo uma function que chama outra function e/ou ela mesma, como é o caso, e faz o fatorial de uma forma diferente do exemplo 4, porém com o mesmo resultado.
